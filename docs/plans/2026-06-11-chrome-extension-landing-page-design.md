# Chrome Extension Landing Page Design

## Direction

Use a clean Chrome-inspired visual language: bright white surfaces, soft blue accents, rounded cards, subtle shadows, and generous spacing. Keep dark mode as a complete secondary theme.

## Page Structure

1. Navigation with product identity, section links, theme control, and install CTA.
2. Conversion-focused hero with Chrome Web Store CTA and a real product screenshot.
3. Compact product capability strip.
4. Six-card feature snapshot covering bookmarks, weather, developer activity, focus, search, and customization.
5. Product screenshot gallery showing the workspace tools and customization panel.
6. Three-step installation and setup explanation.
7. Privacy reassurance based on the extension's disclosed data practices.
8. Final install CTA and legal footer.

## Behavior

- All install CTAs open the official Chrome Web Store listing.
- Navigation links scroll to page sections.
- The mobile navigation collapses behind a menu button.
- Theme selection persists through the existing local storage key.
- Existing privacy and terms routes remain available.

## Verification

- Run the production build.
- Inspect desktop and mobile layouts in the local browser.
- Confirm theme toggle, mobile menu, internal navigation, and store links.
