# Developer Workspace Extension

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)
![Build Status](https://img.shields.io/badge/build-passing-brightgreen.svg)

A powerful Chrome extension that transforms your new tab into a productive developer workspace.

## Features

### Clock & Weather
- Customizable digital clock with multiple formats (12/24 hour)
- Multiple timezone support
- Optional seconds display
- Local weather display with temperature unit toggle (°C/°F)

### Smart Bookmarks
- Integration with Chrome bookmarks
- Category-based organization (Development, AI, Social, etc.)
- Custom folders support
- Fast search and filtering
- Most visited sites view with visit count
- Favicon auto-detection

### Developer Tools Panel
- GitHub activity tracking
- Real-time API status monitoring
- Quick documentation access
- Pomodoro timer with notifications
- Terminal-style notes

### Customization
- Dark/Light mode
- Multiple background options
- Custom background upload
- Collapsible sections
- Component visibility toggles

### Chrome Integration
- Chrome bookmarks sync
- Most visited sites tracking
- History integration
- Search with Chrome's engine
- Cross-device sync

## Installation
1. Clone the repository
```bash
git clone https://github.com/yourusername/developer-workspace.git
```

2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extension directory
5. The extension will replace your new tab page

## Usage

### Bookmarks Management
- Click "+" to add new bookmarks
- Create folders for organization
- Use the search bar to filter bookmarks
- Toggle between bookmarks and most visited sites
- Categories for easy access

### Developer Features
- Connect your GitHub account to see activity
- Monitor popular API statuses
- Quick access to documentation
- Use Pomodoro timer for focused work
- Take notes in terminal style

### Settings
- Access settings via the gear icon
- Customize appearance and functionality
- Toggle features on/off
- Reset to defaults if needed

## Technical Notes
- Uses Chrome Storage Sync API for cross-device synchronization
- Implements Chrome Bookmarks API for seamless integration
- Utilizes Chrome History API for visit tracking
- Built with vanilla JavaScript for optimal performance
- Responsive design for all screen sizes

## Browser Support
- Chrome (Latest)
- Chromium-based browsers (Edge, Brave, etc.)
